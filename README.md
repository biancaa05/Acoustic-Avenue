# 🎸 Acoustic Avenue | Showroom Premium de Instrumente Muzicale

**Acoustic Avenue** este o platformă web de tip showroom, dedicată unui magazin boutique de instrumente muzicale. Proiectul pune accent pe o estetică "fancy" și minimalistă, combinând tipografia de lux cu layout-uri avansate în CSS Grid și elemente interactive unice.

---

## ✨ Caracteristici Principale

* **Identitate Vizuală de Lux:** O paletă cromatică sofisticată compusă din **Mahon (#3A1F1B)** și **Auriu (#C5A059)**, susținută de fonturi premium precum *Bodoni Moda* și *Pinyon Script*.
* **Grid Layout Dinamic:** O arhitectură custom **CSS Grid 5x6** (bazată pe o schiță tehnică precisă) care creează o experiență de navigare artistică și non-liniară.
* **Pin de Locație "Motorizat":** Integrare personalizată Font Awesome cu un pin rotit la -45°, dotat cu o animație de vibrație mecanică de înaltă frecvență care "iese" din cadrul pozei de prezentare.
* **Meniu "Hamburger" Responsive:** Sistem de navigare mobil implementat 100% în CSS (folosind *Checkbox Hack*), asigurând o experiență fluidă fără dependențe de JavaScript.
* **Glassmorphism UI:** Secțiuni care utilizează efecte de blur (*backdrop-filter*) și straturi semi-transparente pentru a oferi profunzime și eleganță.

---

## 🛠️ Tehnologii Utilizate

* **HTML5:** Structură semantică pentru SEO și accesibilitate.
* **CSS3 (Advanced):**
    * **CSS Grid Areas:** Control precis asupra layout-ului complex pe 5 coloane.
    * **Animații Keyframes:** Efecte motorizate pentru pin-ul de locație și interacțiuni de tip hover.
    * **Flexbox:** Alinierea componentelor interne și a elementelor din header.
* **Font Awesome 6:** Iconografie scalabilă de înaltă calitate (Guitar, Piano, Drum, Map-pin).
* **Google Fonts API:** Integrarea fonturilor serif și script pentru un aspect grandios.

---

## 📱 Design Adaptabil (Responsive)

Proiectul este complet optimizat pentru orice dimensiune de ecran. Folosind **Media Queries**, grid-ul complex de pe desktop se reconfigurează automat într-un flux vertical pe o singură coloană pentru dispozitivele mobile, garantând funcționalitatea magazinului pe orice device.
