# Acoustic-Avenue
🎸 Acoustic Avenue | Premium Musical Instruments Showroom
Acoustic Avenue is a high-end, responsive web platform designed for a boutique musical instruments store. This project focuses on a "fancy," minimalist aesthetic, combining luxury typography with advanced CSS Grid layouts and interactive elements.

✨ Key Features
Bespoke Visual Identity: Utilizing a sophisticated color palette of Mahogany (#3A1F1B) and Gold (#C5A059), paired with high-end typography (Bodoni Moda and Pinyon Script).

Dynamic Grid Layout: A custom-engineered 5x6 CSS Grid architecture that creates a non-linear, artistic showroom experience.

Interactive "Motorized" Map Pin: A custom Font Awesome integration featuring a map pin tilted at -45° with a high-frequency "mechanical" vibration animation to highlight the showroom's physical location.

Responsive "Hamburger" Menu: A 100% CSS-driven mobile navigation system (zero JavaScript dependency) for seamless browsing on tablets and smartphones.

Glassmorphism UI: Sections utilize backdrop-filter blurs and semi-transparent layers to maintain a sense of depth and luxury.

🛠️ Technologies Used
HTML5: Semantic structure for better SEO and accessibility.

CSS3 (Advanced):

CSS Grid Areas: Precise control over the complex 5-column layout.

Custom Keyframe Animations: Powering the "motorized" effects and interactive hovers.

Flexbox: Used for navigation and internal component alignment.

Font Awesome 6: For high-quality, scalable iconography.

Google Fonts API: Serving premium serif and script typefaces.

📱 Responsive Design
The project is fully optimized for all screen sizes. Using CSS Media Queries, the complex desktop grid automatically reconfigures into a streamlined, single-column vertical flow for mobile devices, ensuring the "Acoustic Avenue" experience is never compromised.
